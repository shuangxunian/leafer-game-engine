# leaferGame v0.25.2

## Version Positioning

`v0.25.2` is the second implementation pass in the `0.25.x` second playable example stage:

**Collect Stars Gameplay Loop Baseline**

This version turns `examples/collect-stars` from a shell into a minimal playable collection loop. The goal is to prove a second example can reuse existing engine/package APIs for movement, HUD, GameFlow, collision, and tile-backed runtime visuals without adding new framework abstractions too early.

---

## Product Boundary

This remains frontend 2D game engine package work.

`v0.25.2` does not add a visual editor, example launcher product, level editor, asset manager, template gallery, online leaderboard, persistence, publishing workflow, or content authoring surface.

It only adds gameplay code inside the second example.

---

## Goals

After `v0.25.2`, `collect-stars` should support:

- ready / running / ended phases through `GameFlow`
- semantic movement and confirm actions through `InputActionMap`
- keyboard bridge attachment from the browser boot path
- normalized player movement inside a bounded playfield
- runtime star spawning
- collision-based star collection
- score and countdown HUD updates
- restart from ended state
- package-boundary coverage proving the second example uses existing engine APIs

---

## Scope

This version includes:

- collect-stars input action map
- player movement component using `limitMovementVector(...)`
- collect-stars gameplay system
- star entity creation with render, size, transform, view, and collider components
- collision query consumption through `CollisionSystem`
- HUD status updates through `createHudText(...)`
- README updates that describe the baseline loop

This version does not include:

- hazards or enemy AI
- audio cues
- sprite animation clips
- browser tooling panel wiring
- persisted high scores
- online services
- generic framework gameplay-loop abstractions

---

## Semantics

- Press confirm to start from `ready`.
- Move with WASD or arrow keys.
- Collecting a star increases score and respawns the star.
- The run ends when the countdown reaches zero.
- Press confirm again to restart after `ended`.
- The example remains a downstream-style consumer of the engine package.

---
