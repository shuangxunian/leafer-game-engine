# leaferGame v0.3.3

## 版本定位

`v0.3.3` 是 `0.3.x` 的第三批落地：

**Prefab and Factory Pass**

这一版的目标不是一步做到完整数据驱动 prefab，而是先把“重复实体组装”从 sample 代码里抽出来，变成明确的工厂能力。

---

## 版本目标

`v0.3.3` 完成后，至少要做到：

- sample 不再手写一整段 player / hazard entity assembly
- framework 提供轻量可复用 factory 抽象
- 后续第二个 sample 可以沿用同一类组装方式

---

## 范围边界

这一版要做：

- 轻量 factory 抽象
- sample 中 player/hazard 工厂化
- 最小测试覆盖

这一版不做：

- prefab JSON/schema
- 面向编辑器的 template 或可视化内容编辑能力
- 复杂序列化/反序列化

---

## 建议语义

- factory 负责组装 entity 和其组件
- factory 可以消费 renderAdapter / renderScene / scene 等上下文
- factory 返回已完成组装的 entity

---

## 完成标准

- player 和 hazard 的组装逻辑从 sample 主流程中抽出
- sample 行为保持一致
- framework 里有可复用的工厂抽象，而不只是 example 局部 helper
