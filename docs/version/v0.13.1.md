# leaferGame v0.13.1

## Version Positioning

`v0.13.1` is the first `0.13.x` delivery pass:

**Input Action Mapping Baseline**

This version adds a small, Node-safe model for mapping physical keyboard inputs to semantic gameplay actions.

---

## Product Boundary

This remains frontend 2D game engine package work.

`v0.13.1` does not add editor UI, keybinding settings screens, persistence, gamepad adapters, touch gestures, browser-only control panels, or npm publishing. It only adds reusable input mapping primitives that downstream games can consume.

---

## Goals

After `v0.13.1`, the project should support:

- semantic action ids such as `move:left`, `jump`, or `pause`
- keyboard bindings that normalize keys consistently with the browser keyboard bridge
- registering and replacing action definitions
- binding and unbinding keyboard inputs
- querying whether an action is currently pressed
- querying whether an action was just pressed this frame
- listing action definitions without exposing mutable internals
- Node-side tests and package export coverage

---

## Scope

This version includes:

- `InputActionMap`
- `defineKeyboardBinding(...)`
- keyboard binding normalization helpers
- `isPressed(...)` and `wasPressed(...)` action queries against `InputSystem`
- tests for binding normalization, duplicate handling, action queries, and immutable list output
- README/public API progress updates
- `v0.13.0` roadmap linkage
- regression verification

This version does not include:

- gamepad bindings
- touch bindings
- rebinding UI
- local storage or profile persistence
- browser panel integration
- example gameplay changes
- npm publishing

---

## Semantics

- `InputActionMap` is Node-safe and does not depend on DOM APIs.
- Keyboard bindings use the same lowercase normalization as `BrowserKeyboardBridge`.
- Action queries are read-only and do not mutate `InputSystem`.
- Duplicate bindings for the same action are ignored.
- Registering an existing action replaces its binding list.
- Missing actions query as not pressed instead of throwing.
