# leaferGame v0.29.5

## Positioning

`v0.29.5` closes the responsive Web runtime stage:

**Responsive Web Runtime Stage Closeout**

No new public package API is added in this closeout. The stage now has enough
runtime, adapter, framework, and example evidence to treat `0.29.x` as complete
for the current 1.0 path.

## Closed Scope

The `0.29.x` stage leaves these package-facing pieces in place:

- `RenderScene.resize(width, height)` updates render scene viewport state.
- `createRenderSceneViewport(...)` and `getRenderSceneViewport(...)` expose
  copied Node-safe viewport metadata for tests and tooling.
- `createBrowserResizeBridge(...)` observes a browser mount/container and calls
  the render scene resize contract.
- `createBrowserRuntime({ resize: true })` can attach the resize bridge and
  detach it during `runtime.stop()`.
- `createBrowserPointerLocalPositionResolver(target)` and
  `getBrowserPointerLocalPosition(event, target)` read current mount bounds for
  pointer-local coordinates after resize.
- The shared example route and standalone `dodge-blocks` entry consume the
  opt-in resize bridge through package-style runtime setup.

For DPR / high-density display behavior, the stage documents the package
boundary as CSS-pixel viewport synchronization: the resize bridge reads
container dimensions and updates render scene viewport metadata, while the
native Leafer renderer owns any backing-store / device-pixel-ratio details.
The package does not introduce a separate DPR scaling API in this stage.

## Boundary

This closeout does not add a responsive page builder, layout system, polished
launcher, gallery, design system, mobile app shell, WeChat SDK wrapper,
publishing workflow, marketplace, account system, ads, monetization, visual
layout editor, device compatibility matrix, or game-specific responsive rules
in the package.

## Next 1.0 Gap

The next planned 1.0 gap is `0.30.x` drag/drop and selection hardening for
pointer-first puzzle interactions such as sorting, pouring, matching, and
source-target actions.
