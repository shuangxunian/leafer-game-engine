# leaferGame v0.27.2

## Version Positioning

`v0.27.2` is the second implementation pass in the `0.27.x` pointer-first puzzle interaction stage:

**Rectangle Hit Testing And Entity Picking Baseline**

`v0.27.1` added pointer position state. This version lets downstream games map a point to entities using runtime rectangles derived from ECS components.

This is for pointer-first mini-game runtime interaction, not editor picking.

## Scope

This version includes:

- pure point-in-rect hit testing
- entity hit rectangles from `TransformComponent + SizeComponent`
- optional collider-backed hit rectangles and layer filtering
- ordered entity hit results
- top entity picking from the end of an entity list
- tests for point, entity, collider, filtering, copied rects, and invalid input

This version does not include:

- drag/drop state
- gesture recognition
- multi-touch tracking
- visual editor selection handles
- scene graph mutation UI
- puzzle-specific rules
