# leaferGame v0.3.2

## 版本定位

`v0.3.2` 是 `0.3.x` 的第二批落地：

**Camera MVP Pass**

这一版的目标不是一次把 camera 做到很重，而是先让它从“数据壳子”变成“真的会驱动 world layer 的系统”。

---

## 版本目标

`v0.3.2` 完成后，camera 至少应该具备：

- 对 `world` layer 的真实平移映射
- 对 `world` layer 的真实缩放映射
- 基于 viewport 的居中语义
- 最小的 follow 能力

---

## 范围边界

这一版要做：

- `RenderScene` 提供 viewport 尺寸
- `CameraSystem` 驱动 world layer
- `moveTo`
- `setZoom`
- 最小 `follow`
- 测试覆盖

这一版不做：

- dead zone
- damping / smoothing
- bounds clamp
- parallax
- 多 camera

---

## 建议语义

- camera 的 `x / y` 表示当前关注的世界坐标点
- `world` layer 会被移动到让该坐标落在 viewport 中心
- `zoom` 作用在 `world` layer 上
- UI / overlay layer 不跟随 camera 变化

---

## 完成标准

- world layer 会随着 camera 的移动和缩放而变化
- follow target 变化时，camera 能更新 world layer 映射
- HUD 和 overlay 不会被 camera 一起缩放/位移
